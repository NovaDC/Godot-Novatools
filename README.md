# Godot-Novatools

A collection of base classes and static functions used in multiple plugins made by NovaDC.

Currently for Godot v4.4 and above.
Backports for a limited set of functions are available for Godot v4.3 in [a branch](https://github.com/NovaDC/Godot-Novatools/tree/4.3_compat).
Godot v4.2 is no longer supported.

Depending on the plugins using this dependency, it may not be required for this plugin to be enabled in order to function. This is on a case by case basis, reference the relevant plugin for more information.
